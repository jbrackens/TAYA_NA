================================================================================
                         KEY CODE IMPORTS
================================================================================

To use the caching layer in your code:

import "phoenix-revival/gateway/internal/cache"

Creating a cached repository:
  redisClient, err := cache.NewRedisClientFromEnv()
  cachedRepo := cache.NewCachedReadRepository(underlying, redisClient)

Creating an invalidation handler:
  invalidator := cache.NewInvalidationHandler(cachedRepo)

Getting metrics:
  metrics := cachedRepo.Metrics()
