## hombrew-idfx-cli

Hombrew tap repo that is used to download `idfx` binary from homebrew.

Contents here are pushed automatically from [idfx-cli](https://github.com/flipadmin/idfx-cli) repo

Any direct commits to the [hombrew-idfx-cli](https://github.com/flipadmin/homebrew-idfx-cli/tree/main) repo, will be overwritten.
