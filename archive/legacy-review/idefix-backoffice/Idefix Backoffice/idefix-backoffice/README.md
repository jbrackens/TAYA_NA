

# Idefix Backoffice

This project was generated using [Nx](https://nx.dev)

Platform includes 4 backoffice apps - `idefix`, `affmore`, `affmore-affiliate`, `campa`

**There are some of the commands that you can use:**

  - `yarn dev:<app>` - launch one of backoffice apps in dev mode
  - `yarn build:<app>` - build one of backoffice apps, build folder -> `dist/apps/<app>`
  - `yarn start:<app>` - start one of backoffice apps in production mode
  - `yarn build:all` - build all backoffice apps
  - `yarn build:all:parallel` - build all backoffice apps in parallel mode
  - `yarn build:all:analyze` - build all backoffice apps with `stats.json` file for analyzing bundle size
  - `yarn analyze:<app>` - run webpack bundle analyzer for the one of backoffice apps
  - `yarn test` - run all tests in backoffice apps.
