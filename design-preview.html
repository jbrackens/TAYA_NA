Icon Standards
Library: Lucide Icons (lucide-react for React components)
Base size: 18px sidebar, 20px icon tiles, 16px inline, 24px page headers
Stroke width: 1.75px (default) — slightly thinner than Lucide's 2px default for a refined feel
Color: Inherit from text color. Active states use brand green. Never use multi-color icons.
Icon tiles: 40x40px container with soft-fill background and brand-border. Radius matches app context.
